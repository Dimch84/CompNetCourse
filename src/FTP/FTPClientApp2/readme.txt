.Net FtpExplorer sample
=======================
     This sample is a Windows Forms-based application using System.Net.FtpWebRquest class. It demonstrates a type of Windows Explorer type of User Interface for browsing FTP sites. Implementation supports most of the FTP Methods: ListDirectory, DownloadFile, UploadFile, MakeDirectory, RemoveDirectory, and so on. 

This sample also demonstrates the implementation of UI-based user interactive custom server certificate validation when connecting to the FTP site using SSL.


Sample Language Implementations
===============================
     This sample is available in the following language implementations:
     C#


To build the sample using the command prompt:
=============================================
     1. Open the Command Prompt window and navigate to the CS subdirectory under the FtpExplorerSample directory.
     2. Type msbuild FtpExplorerCS.


To build the sample using Visual Studio:
=======================================
     1. Open Windows Explorer and navigate to one of the language-specific subdirectories under the FtpExplorerSample directory.
     2. Double-click the icon for the .sln (solution) file to open the file in Visual Studio.
     3. In the Build menu, select Build Solution.
     The application will be built in the default \bin or \bin\Debug directory.


To run the sample:
=================
     1. Navigate to the directory that contains the new executable, using the command prompt or Windows Explorer.
     2. Type FtpExplorerCS.exe at the command line, or double-click the icon for FtpExplorerCS.exe to launch it from Windows Explorer.
